# Launch Event Plan

## Format

Thirty-minute live demo with three moments:

1. Fast calculator demo.
2. Sync, AR fallback, and beta-feedback walkthrough.
3. Community challenge reveal.

## Run of Show

| Minute | Segment | Owner |
|---:|---|---|
| 0 | Open with neon calculator result | host |
| 5 | Show settings, sync, beta, and community controls | host |
| 12 | Trigger AR fallback and gesture controls | host |
| 18 | Launch the daily challenge | host |
| 24 | Q&A and install CTA | host |

## Success Metrics

- Demo completion rate.
- App-store page visits.
- Community challenge shares.
- Beta feedback submissions.
